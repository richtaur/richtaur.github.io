# richtaur.com

The personal website of [Matt Hackett](https://twitter.com/richtaur), a game developer in California.

![Matt Hackett](http://i.imgur.com/LZAGr6y.png)

## Run

`npm install`
`make`
`make serve`

## Publish

`make deploy`

## Add a new post

* Create a new file, e.g. `./src/post/example-url.md`
